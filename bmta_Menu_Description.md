### 4. 내정보
로그인을 안한 사용자는 하단 네비게이션 바의 내정보를 클릭하면 로그인 페이지로 이동된다.

로그인한 사용자인 경우 페이지에 가장 상단에는 동그란 프로필 이미지와 닉네임이 보여져
바로 아래에는 현재 나의 포인트가 몇인지 볼 수 있어
그리고 쪽지함이라는 메뉴와 설정, 공지사항, 고객센터 그리고 가장 마지막에는 로그아웃 버튼과 버전 정보가 있다.

쪽지함은 메뉴 클릭하면 보낸 쪽지함과 받은 쪽지함 전부를 확인할 수 있는 화면과 보낸 쪽지함, 받은 쪽지함을 필터로 구분할 수 있도록 해줘
쪽지함 메뉴에는 제목만 최대 한 줄로 보여지고 누가 언제 몇 시에 보냈는지 알 수 있어
클릭하면 상세 페이지로 이동하여 내용을 볼 수 있지 또 상세 페이지에서는 답장보내기 및 사용자 프로필을 선택해서 차단 혹은 신고 기능도 들어가면 좋겠다

설정 메뉴는 클릭하면 프로필 사진 변경 내 닉네임이 나오고 아래 닉네임 변경 버튼이 있고 비밀번호 변경, 이메일 변경, 알림 설정 이렇게 3개 메뉴가 있어
닉네임 변경 버튼을 클릭하면 나의 닉네임이 입력된 채 수정할 수 있도록 입력필드가 활성화되면 좋겠고 닉네임 변경 버튼은 확인 버튼으로 변경돼어야해

그리고 비밀번호 변경 메뉴를 클릭하면 변경 페이지로 이동되고, 현재 비밀번호 확인란과 새 비밀번호 입력 필드 그리고 새 비밀번호 확인 입력 필드 3개 필드와 변경 버튼이 보여져

다음으로 이메일 변경 버튼 클릭 시 닉네임 변경과 동일하게 현재 이메일이 수정 가능한 상태로 활성화되고 수정할 수 있어
이메일로 우리는 사용자 인증을 하기때문에 이메일 변경 버튼은 인증하기 버튼으로 변경되고 버튼을 클릭하면 이메일 필드는 비활성화되고 입력된 이메일로 인증번호 숫자 6자리가 발송돼 (이건 회원가입 시 동일한 인증 방식)
그리고 인증번호를 입력할 수 있는 필드가 추가로 보여지고 확인 버튼도 함께 보여져
인증번호가 맞다면 이메일 필드 아래에 변경되었습니다. 라는 마이크로텍스트가 보여지고 인증번호가 틀리면 인증번호를 다시 확인해 주세요. 라는 문구가 보여져

알림 설정 같은 경우는 알림 ON/OFF 토글 스위치가 있어
디바이스 설정 메뉴로 이동해서 알림을 키도록 하는건 나중에 하자

공지사항 메뉴는 게시판 형식으로 공지사항 제목이 보여지고 년도와 월일이 있어
클릭하면 공지사항의 상세 내용을 확인할 수 있지

마지막으로 고객센터 메뉴는 메뉴 UI만 만들어놓고 클릭 시 어떻게 보여줄지는 좀 만 생각하자

로그아웃 버튼 클릭하면 로그아웃되고 브타 피드 화면으로 이동돼
